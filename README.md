# Dimensionality Reduction

These materials are from a workshop through [SFU's Institute for Neuroscience and Neurotechnology](https://www.sfu.ca/neuro-institute.html) that was held in October, 2025 on common dimensionality reduction techniques including principal components analysis, independent components analysis, and singular value decomposition. 

[The video link to the workshop is here](ADD LINK) and the slides and code files are in this repository in both MatLab and R. For regular R users: there are m a n y ways to run these tests in R. Here, I went with the one closest to working in base R for the R newbies.
